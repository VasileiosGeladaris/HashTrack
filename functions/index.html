HashTrack is an API that lets you track anything on a map with real-time updates


It can be used by any business delivering a product.


The two key features are:
- Providing the client with a map showing the driver's location, getting real-time updates
- Providing the driver with an optimized route to the client

Checkout our GitHub Page for instructions on how to use it and further details.
